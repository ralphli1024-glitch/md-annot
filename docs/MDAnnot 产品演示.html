特点

给 Markdown 文字加三层标记 + 文本锚定定位 + 匹配度识别：高亮、划线、评论，不改原文

📄

不污染源文

支持在阅读时对文字进行高亮、划线和评论，所有批注数据独立存储在 .annotations/ 目录，不污染 .md 源文件。

📋

批注列表

右侧面板集中管理当前文档所有批注，支持按类型筛选、按位置排序，一目了然。

🎨

颜色配置

12 种预设色卡快速选色，高亮、划线、评论三种颜色独立配置，满足不同标注习惯。

📤

导出批注

支持导出当前文档批注和所有文档批注为 Markdown 格式，方便分享和备份。

📍

批注定位

批注不依赖行号，基于文本内容指纹（FNV-1a 哈希 + 上下文）定位。编辑文档后自动匹配，不会因增加删改行导致批注错位。

📊

匹配度识别

四级降级匹配引擎：精确匹配 → 模糊匹配 → 段落匹配 → 标记丢失。每条批注右侧显示 10 格彩色置信度进度条，绿色到红色渐变，一眼判断锚定状态。

核心功能

高亮 / 划线 / 文字评论 + 文本锚定，覆盖 Markdown 批注场景

1

编辑和阅读模式批注

在编辑模式选中文字 → 弹出浮动工具栏 → 创建高亮/划线/评论。也在阅读模式直接标注，两种模式渲染效果一致。批注密集时按钮自动高亮已有批注，点击即 toggle 删除。

批注数据存储在独立的 `.obsidian/md_annot/` 目录，配合 Remotely Save 开启配置文件夹同步即可跨设备自动同步。

批注列表面板

批注列表集中管理当前文档所有批注。点击任意批注卡片自动定位到文档中对应文字位置并闪烁提示，支持按类型筛选、按位置排序、一键删除。

5

多设备同步

批注数据存储在 .obsidian/md_annot/ 目录，不依赖第三方服务，完全本地优先。

配合 Obsidian 生态的同步方案（Remotely Save + OneDrive / WebDAV 或 Obsidian Sync），在设置中开启「同步配置文件夹」后，批注数据随 Obsidian 配置一起自动同步到所有设备。

无需手动导出导入，无需额外配置。Mac 上标注 -> 手机上看 -> Win 上继续编辑，批注始终一致。

文本锚定引擎

不依赖行号，基于文本内容指纹定位批注位置。编辑后自动降级匹配，批注不会轻易丢失。

精确匹配

文本内容完整未修改时，精确定位到字符级位置。内容哈希校验 O(1) 确认。
